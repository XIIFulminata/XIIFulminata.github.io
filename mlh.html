Emoji Home

Emoji Home was built to emulate a home automation system. Users send specific combinations of emojis to their "house" and it carries out certain tasks. Sending would turns a bathroom light on or off, would lock your front door, and would show you who was at your front door.

How we Built It

We used the twilio sms messaging api and node.js with express.js to write a server application that would handle incoming texts and execute the appropriate action and notify the user once the action has completed. We 3D printed a small floor plan and added LED's to visually represent calls to different devices.

Personal Accomplishments

I worked on testing and triggering the sound effects of each component using JavaScript (so now I can make a piano!). I also modeled and printed the floor plan used to represent the home.

Running Total

While working on a previous project, I tried to source parts while comparing prices and keeping track of all my links. When the amazon wish list chrome extension didn't show all the data I wanted, I began to maintain a spreadsheet. It was a hassle, and I wanted a simpler way to keep track of my parts. When users click the extension button while selecting the item name, a popup appears that allows the user to enter a quantity, price, and comments as needed. These are then added to a project page in the form of a dynamic table.

How we Built It

The chrome extension was built as a pop up using JavaScript where the user can highlight the name of the item, click the extension and then it would be auto populated with the name and a default quantity of 1. The information from the popup name, price, and quantity is added as an entry to our projects page which you can access through the extension.

Personal Accomplishments

This was my first experience making a chrome extension. One of our initial ideas centered around building a web scraper, so I got some experience using node.js, and exploring the DOM. I also created the landing page for tables created by the extension to maintain track of the total.
